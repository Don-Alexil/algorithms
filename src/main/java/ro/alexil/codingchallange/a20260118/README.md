# Problem: Split Array Largest Sum
**Time Limit:** 40 min

## Statement
Given an integer array `nums` and an integer `k`, split `nums` into `k` **non-empty subarrays** such that the **largest sum** among these subarrays is **minimized**.

Your task is to find this minimized largest sum by choosing the optimal way to split the array.

## Constraints
- `1 ≤ nums.length ≤ 10³`
- `0 ≤ nums[i] ≤ 10⁴`
- `1 ≤ k ≤ nums.length`
